# TomTroc

Application PHP MVC pour échanger des livres entre utilisateurs.

## Installation

- Cloner le dépôt
- Importer le fichier SQL P4-Tom-Troc.sql
- Lancer le serveur PHP intégré : `php -S localhost:8000 -t public`

## Connection utilisateur

- Les adresses email des 5 différents utilisateurs créé son dans le fichier P4-Tom-Troc.sql aux lignes 143 à 148
- Les mots de passes sont tous identiques : 12345678
